OUGC Show Self Unaproved README
-----------------

Requirements:
 - MyBB version 1.6.4+
 - PluginLibrary version 6+

Installation:
 - Upload the contents of the "upload" folder to your MyBB forum root installation.
 - Activate plugin from the MyBB forum administrator panel.
 - Apply core edits to files from the plugin information view.

Uninstallation:
 - Deactivate plugin from the MyBB forum administrator panel.
 - Remove plugin files from your MyBB forum installation.
